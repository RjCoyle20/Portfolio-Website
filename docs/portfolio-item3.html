Number Guesser

A full-stack react web game

Number Guesser is a web based game of luck and skill. Set your number of attempts and figure out which random number the CPU has divined. Play the odds or go with your gut!

Choose to login to save your past games in a persistent database or play the game without logging in.

Utilizing Spring, React and MySQL, this game is simple, fun and challenging.
