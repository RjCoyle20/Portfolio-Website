Software for the management of business processes

Build-A-Beer is a fun app for craft beer brewers and enjoyers! Fill out a brief form and see what mystery beer gets created for you and find recommendations for similar real-life brews.

Using a call to Punk API, Build-A-Beer generates a custom brew for each user, which the user can name, or they can have another call to the API generate a name for them!

The beer is then saved to the user's "Fridge," where they can edit or delete it. Also, you can view other users' fridges!
